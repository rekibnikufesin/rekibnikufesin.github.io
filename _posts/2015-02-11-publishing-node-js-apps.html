There are many ways to publish a node.js app- each with their own pros and cons. In this 9 minute video, I show you how to quickly publish your node app using github and running it with a utility app called screen.
Take aways:
Configuring your server to pull your code from github
Launch your app with screen so that it stays running after you log out
Make changes in your local dev environment
Merge those changes into production after testing
Avoid writing code on production servers</p>
